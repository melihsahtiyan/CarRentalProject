# Car Rental Project
Rent A Car Project is a car hiring system developed with SOLID principles and built with these techniques: 
+ C#
+ ASP.NET
+ .NET Core
+ AOP
+ Autofac
+ Entity Framework
+ JSON Web Token
+ SQL SERVER
